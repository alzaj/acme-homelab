# Ticket #0002

## Title
Deploy Apache Web Server

## Background

Zum Testen einer neuen Webanwdndung ist ein Web-Server erforderlich

## Requirements

- Installation von Apache
- einfache Seite index.html erstellen
- die Seite index.html muss enthalten:
  - "Welcome to Acme GmbH"
  - hostname
  - aktuelle Zeit und Datum beim Aufruf der Webseite.

## Acceptance Criteria

- Autostart des Apache-Webservers beim System-Start
- HTTP-Port ist erreichbar.
- Die Testseite wird ausgeliefert.
- Erreichbarkeit der Webseite von einem anderen Rechner im LAN.
